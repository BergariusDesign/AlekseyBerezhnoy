как я могу регулировать размер окон на главной страницы с проектом?

В вашем коде размер окон (карточек) регулируется с помощью CSS Grid системы. Вся сетка поделена на 12 колонок.

Чтобы изменить размеры, найдите в блоке <style> раздел HERO & GRID (примерно строки 80-100). Там есть три главных класса, которые управляют размерами:

1. Как изменить ШИРИНУ (Горизонтальный размер)
Ширина задается параметром grid-column: span X, где X — это количество колонок, которые занимает карточка (всего их 12).

Найдите эти строки в CSS:

CSS
.col-8 { grid-column: span 8; height: 500px; } /* Занимает 8 из 12 (примерно 66% ширины) */
.col-4 { grid-column: span 4; height: 500px; } /* Занимает 4 из 12 (примерно 33% ширины) */
.col-6 { grid-column: span 6; height: 350px; } /* Занимает 6 из 12 (50% ширины, половина) */
Если хотите сделать карточку на всю ширину, поставьте span 12.

Если хотите три одинаковые карточки в ряд, создайте класс с span 4 (так как 4+4+4=12).

Важно: Сумма span в одной строке должна быть равна 12 (или меньше, тогда останется пустое место). Если сумма больше 12, карточка перепрыгнет на следующую строку.

2. Как изменить ВЫСОТУ (Вертикальный размер)
Высота задается параметром height в пикселях.

В том же месте кода:

CSS
.col-8 { ... height: 500px; } /* Высокая карточка */
.col-6 { ... height: 350px; } /* Карточка пониже */
Просто поменяйте 500px на любое другое число, например 600px (выше) или 250px (узкая полоска).

3. Как применить новые размеры к конкретной карточке?
В HTML-коде (внизу файла, внутри <div class="grid">) у каждой карточки написан её класс размера:

HTML
<div class="card col-8" ...>
Если вы хотите сделать её узкой, просто поменяйте класс в HTML:

HTML
<div class="card col-4" ...>
4. Адаптивность (Мобильные телефоны)
Обратите внимание на этот кусок кода чуть ниже:

CSS
@media (max-width: 1024px) { 
    .col-8, .col-4, .col-6 { grid-column: span 12; height: 400px; } 
    ... 
}
Он означает: "На экранах меньше 1024px (планшеты и телефоны) игнорируй мои настройки и делай все карточки на всю ширину (span 12) и высотой 400px". Если хотите менять размер и на телефонах, правьте цифры именно внутри этой скобки @media.
